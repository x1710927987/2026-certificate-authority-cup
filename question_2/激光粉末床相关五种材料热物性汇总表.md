# 激光粉末床相关五种材料热物性汇总表

## 数据来源与不确定性说明

表中 “熔点 / 熔融温度、比热容 (cp)、密度、线膨胀系数 (CTE)” 优先取自厂商材料数据表；当不同来源、温区或状态（热处理、wrought vs. LPBF/SLS 成形件、聚合物含水 / 配方）存在差异时，以 min–max 范围给出，并在参考列标注原文要点与单位换算规则。

## 数据与标准说明

主要来源包含 EOS、Renishaw、Evonik 等设备 / 材料厂商官方数据表、MatWeb 材料数据库、ASM 金属手册、NETZSCH 热物性测试报告，以及 ASTM 国际测试标准（用于识别数据表中的测试标准与温区，如 ASTM E228）。

## 单位换算规则

- 1 J/g·K = 1000 J/kg·K
- 1 µm/(m·K) = 1×10^-6/K
- 若原文为 10^-5 in/in・°F（如 Evonik 给出的 CTE 表达），换算到 10^-6/K 需先 ×10（变成 10^-6/°F）再 ×1.8（°F→K）

## 行级来源注释

1. **PA12**：同为 “PA12” 但数据表可能指 SLS 成形件（如 0.93 g/cm³）或树脂 / 块料（约 1.01–1.04 g/cm³），且 CTE 与 cp 受配方 / 含水 / 测试方法影响较大。
2. **316L**：本表 CTE 与 cp 优先采用 0–100°C 附近典型值；同一材料在更高温区 CTE 会增大，因此数据表常提供 “平均 CTE (某温区)” 与多温区表。
3. **Ti-6Al-4V**：比热为温区依赖量；Upmet 给出随温度升高 cp 显著上升（例如 0.930 J/g・K@870°C），故主表列保留 20°C 附近范围并提示温区。
4. **AlSi10Mg**：EOS 与 Renishaw 均给出 0–100°C 附近 CTE≈20×10^-6/K，但 EOS 同表对 25–300°C 给到 27×10^-6/K；比热 EOS 旧表区分 as built 与 heat treated，故标注 “状态 / 热处理依赖”。
5. **IN718**：不同数据表可能给出 wrought 典型值或 LPBF 成形件密度下限；VDM 提供 “随温度的 cp/CTE 表”，而 Renishaw 给的是 25–760°C 区间 CTE 范围，故主表列以室温附近典型范围汇总。
